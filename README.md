# Tiny Commands

**Completing various tasks will be faster, thanks to tiny versions of Minecraft commands.**  
For example, you no longer need to write _**/time set day**_, this action will replace by the command _**/day**_.

</div>

```
Note: This is a client-side mod. You do not need to install this on servers to use it.
```

<div align="center">

---

## 🕒 Time & Weather

| Tiny commands | Standard commands | Description |
| :---: | :---: | :---: |
| **/day** | /time set day | Sets time to day |
| **/night** | /time set night | Sets time to night |
| **/noon** | /time set noon | Sets time to noon |
| **/midnight** | /time set midnight | Sets time to midnight |
| **/sun** \| **/wtr clear** \| **/wtr c** | /weather clear | Sets weather to clear |
| **/rain** \| **/wtr rain** \| **/wtr r** | /weather rain | Sets weather to rain |
| **/thunder** \| **/wtr thunder** \| **/wtr t** | /weather thunder | Sets weather to thunder |

## 🎮 Gameplay & Stats

| Tiny commands | Standard commands | Description |
| :---: | :---: | :---: |
| **/gm 0** | /gamemode survival | Sets gamemode to survival |
| **/gm 1** | /gamemode creative | Sets gamemode to creative |
| **/gm 2** | /gamemode adventure | Sets gamemode to adventure |
| **/gm 3** | /gamemode spectator | Sets gamemode to spectator |
| **/diff p** \| **/diff peaceful** | /difficulty peaceful | Sets difficulty to peaceful |
| **/diff e** \| **/diff easy** | /difficulty easy | Sets difficulty to easy |
| **/diff n** \| **/diff normal** | /difficulty normal | Sets difficulty to normal |
| **/diff h** \| **/diff hard** | /difficulty hard | Sets difficulty to hard |
| **/sp** | /spawnpoint | Sets spawn point |
| **/heal** | /effect give @s instant_health | Heals the player |
| **/feed** | /effect give @s saturation | Feeds the player |
| **/nv** | /effect give @s night_vision | Infinite Night Vision |
| **/eclear** | /effect clear @s | Clears all effects |
| **/tphere [player]** | /tp [player] @s | Teleports player to you |

## ⚙️ Gamerule Shortcuts

| Tiny commands | Standard commands |
| :---: | :---: |
| **/gr keepinv [t/f]** | /gamerule keepInventory [true/false] |
| **/gr day [t/f]** | /gamerule doDaylightCycle [true/false] |
| **/gr wtr [t/f]** | /gamerule doWeatherCycle [true/false] |
| **/gr pvp [t/f]** | /gamerule pvp [true/false] |
| **/gr grief [t/f]** | /gamerule mobGriefing [true/false] |
| **/gr rts [number]** | /gamerule randomTickSpeed [number] |
